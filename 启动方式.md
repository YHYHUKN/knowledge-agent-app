# 启动方式

本文档说明项目的所有启动方式与操作步骤。

---

## 方式一：双击运行（推荐）

直接双击项目根目录下的 `.bat` 文件即可运行：

| 文件 | 说明 |
|------|------|
| `start-dev.bat` | 开发模式 — 自动安装依赖 + 启动 `npm run dev` |
| `start-prod.bat` | 生产模式 — 自动安装依赖 + 构建 + 启动 `npm start` |
| `stop.bat` | 停止服务器 — 关闭所有 Node 进程 |

> **启动后访问**：`http://localhost:3000`

---

## 方式二：命令行运行

```bash
# 进入项目目录
cd F:\笔123试\knowledge-agent-app

# 安装依赖
npm install

# 开发模式
npm run dev

# 生产模式（需先构建）
npm run build
npm start
```

---

## 数据说明

- **内置数据**：首次启动自动预置 3 条种子知识资产，无需手动初始化
- **用户数据**：用户新增的资产通过 localStorage 持久化，刷新 / 重启浏览器后保留
- **数据重置**：如需恢复默认 3 条种子数据，请在浏览器中按 `F12` → `Application` → `localStorage` → 删除 `knowledge_assets_v1` 键，刷新页面即可
- **服务重启**：服务端内存（API 检索计算用）重启后仅种子数据保留，用户新增数据不受影响（已持久化在 localStorage）
