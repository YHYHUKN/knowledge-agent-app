# Screenshots

> 关键页面截图，用于 GitHub README 展示。建议 1920×1080、亮/暗主题各一组、文件 < 500KB、PNG 格式。

## 命名规范

```
01-workbench.png         # 主界面（亮色）
02-add-asset.png         # 新增资产弹窗
03-chat.png              # Agent 问答
04-trace.png             # Trace 检索链路
05-dark-workbench.png    # 主界面（暗色，可选）
06-dark-trace.png        # Trace（暗色，可选）
```

## 录屏建议

| 工具 | 平台 | 推荐场景 |
|------|------|---------|
| **ScreenToGif** | Windows | 极简录屏，导出 GIF/MP4 |
| **ShareX** | Windows | 截图+录屏一体 |
| **OBS Studio** | Windows/Mac/Linux | 录制更长视频 |

**demo.gif 录制建议**：
- 宽度 800-1000px（GitHub README 最佳显示宽度）
- FPS 15-20（文件小且流畅）
- 时长 30-45 秒
- 文件大小 < 5MB
